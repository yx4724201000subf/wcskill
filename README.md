# wcskill

Ounin 的内容创作工具箱。做成 Claude Code skill，给创作者用。

---

## 工具箱


| Skill           | 做什么                           |
| --------------- | ----------------------------- |
| `/wc`           | 主入口，自动路由到对的工具                 |
| `/wc-xhs-title` | 小红书标题引擎。从 62 套验证过的标题结构中匹配最优方案 |
| `/wc-hook-dy`   | 抖音对标开头引擎。拆对标账号表格 → 提炼 0-2s 开头范式库 → 为你的正文产出 5-10 个备选开头 |


---

## 安装

```bash
git clone https://github.com/yx4724201000subf/wcskill.git /tmp/wcskill && mkdir -p ~/.claude/skills && cp -r /tmp/wcskill/skills/wc* ~/.claude/skills/ && rm -rf /tmp/wcskill
```

安装后在 Claude Code / Cursor 中输入 `/wc` 即可。

---

## 使用

```
/wc              → 主入口，告诉你该用哪个工具
/wc-xhs-title    → 直接进入小红书标题引擎
/wc-hook-dy      → 直接进入抖音对标开头引擎
```

### 小红书标题引擎

给一个话题，从 62 套标题结构中自动匹配最合适的 5-8 个，生成定制标题。

- 10 类心理驱动力覆盖（量化锚点、经历叙事、风险警示、反直觉、结果导向、人群锁定、信息差、观点表达、场景互动、权威嫁接）
- 每个标题附结构编号和选用理由
- 输出 Top 3 推荐

### 抖音对标开头引擎

两阶段工作流，只专注 0-2s 注意力劫持区间，不碰选题，不改正文主体。

- **阶段1｜建库**：喂对标账号的「文案+数据」表格（csv/tsv/xlsx），自动按起手类型 × 句子任务 × 整体观感聚类，以该账号自身播放中位数为基准加权筛选，产出 3-7 张范式卡
- **阶段2｜套用**：喂自己的正文，从范式库匹配 5-10 个备选开头（0-2s + 好奇区间 + 承接引子），至少覆盖"标签组合型 + 小半句型"两种起手类型
- 范式库可选本地持久化，后续写新正文直接复用
